# Time complexity:

```bash
access: O(1)

push: O(1) # (can be O(n). Because, due to JS arrays are dynamic in nature under the hood JS moves the entire array to different location to make it flexible.)

pop: O(1)

insert/unshift/splice: O(n)

delete: O(n)

search: O(n)
```
